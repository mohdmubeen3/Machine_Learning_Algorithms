
# Logistic Regression

## 📌 Overview

This project demonstrates the implementation of **Logistic Regression** using **Python** and **Scikit-learn**.

The model predicts whether a student will **Pass** or **Fail** based on the number of hours studied.

---

## 🎯 Objective

To understand and implement Logistic Regression for binary classification and predict both the probability and the final class of a new observation.

---

## 🧠 Algorithm

**Logistic Regression**

Logistic Regression is a supervised machine learning algorithm used for **binary classification**. It predicts the probability of an observation belonging to a class using the **Sigmoid Function**.

---

## 📊 Dataset

This implementation uses a manually created dataset for learning purposes.

### Feature (Independent Variable)

- Hours Studied

### Target (Dependent Variable)

- Pass (1)
- Fail (0)

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
