
# Decision Tree Classification

## Overview

This project demonstrates the implementation of the **Decision Tree Classification** algorithm using Python and Scikit-learn.

A Decision Tree is a supervised machine learning algorithm that predicts outcomes by asking a sequence of decision-based questions. It creates a tree-like structure where each internal node represents a condition, each branch represents the outcome of that condition, and each leaf node represents the final prediction.

---

## Problem Statement

A bank wants to automate its loan approval process. Based on a customer's **Age**, **Annual Income**, and **Credit Score**, the model predicts whether the customer's loan application should be **Approved** or **Rejected**.

---

## Objective

Develop a Decision Tree Classification model that learns from historical customer data and predicts whether a loan application should be approved.

---

## Dataset

The dataset used in this project is a manually created sample dataset containing:

- Age
- Annual Income
- Credit Score
- Loan Approval Status (Yes / No)
