# Simple Linear Regression

## Objective
Predict salary based on years of experience using Simple Linear Regression.

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
1. Load Dataset
2. Visualize Data
3. Train-Test Split
4. Train Model
5. Predict
6. Evaluate
7. Plot Regression Line

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score
