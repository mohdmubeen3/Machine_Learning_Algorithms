# Multiple Linear Regression

## 📌 Overview

This project demonstrates the implementation of **Multiple Linear Regression** using **Python** and **Scikit-learn**.

The model predicts **CO₂ emissions** of a car using multiple input features such as **Engine Size**, **Number of Cylinders**, and **Fuel Consumption**.

---

## 🎯 Objective

To understand and implement Multiple Linear Regression by training a machine learning model that predicts a continuous numerical value using multiple independent variables.

---

## 🧠 Algorithm

**Multiple Linear Regression**

Multiple Linear Regression is a supervised machine learning algorithm that predicts a continuous numerical value using two or more independent variables.

---

## 📊 Dataset

This implementation uses a small manually created dataset for learning purposes.

### Features (Independent Variables)
- Engine Size
- Cylinders
- Fuel Consumption

### Target (Dependent Variable)
- CO₂ Emission

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

---

## 🚀 Workflow

1. Import required libraries
2. Create the dataset
3. Create a DataFrame
4. Select features (X) and target (y)
5. Split the dataset into training and testing sets
6. Create the Multiple Linear Regression model
7. Train the model using the training data
8. Predict CO₂ emissions
9. Evaluate the model using Mean Squared Error (MSE) and R² Score
10. Predict CO₂ emissions for a new car

---

## 📈 Evaluation Metrics

- Mean Squared Error (MSE)
- R² Score

---

## 📌 Conclusion

The Multiple Linear Regression model successfully learned the relationship between multiple input features and CO₂ emissions. The high R² score indicates that the model explains most of the variation in the target variable, making it suitable for predicting continuous numerical values.
