
# K-Means Clustering

## 📌 Overview
This project demonstrates the implementation of **K-Means Clustering** using Python and Scikit-learn.

The model groups similar data points into **K clusters** based on the distance between data points and their cluster centroids.

---

## 🎯 Objective
To understand and implement the K-Means Clustering algorithm for grouping similar data points and visualize the resulting clusters.

---

## 🧠 Algorithm

### K-Means Clustering

K-Means is an **unsupervised machine learning algorithm** used for clustering. It partitions data into **K non-overlapping clusters** by assigning each data point to the nearest centroid and updating the centroids iteratively until convergence.

---

## 📊 Dataset

This implementation uses a **manually created dataset** for learning purposes.

### Features

- Feature 2 (Y-coordinate)

### Number of Clusters

- K = 2
